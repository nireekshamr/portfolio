# Telco Customer Churn Prediction

## Project Overview
This project predicts whether a telecom customer is likely to churn using Machine Learning classification algorithms.

## Algorithms Used
- Logistic Regression
- Random Forest Classifier

## Dataset
- Telco Customer Churn Dataset

## Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Model training
- Model evaluation
- ROC Curve
- Confusion Matrix
- Classification Report
- Feature Importance

## Project Files
- churn_prediction.ipynb
- churn_model.pkl
- requirements.txt
- report-task1.pdf
- README.pdf

## Requirements

Install the required libraries:

```bash
pip install -r requirements.txt
```

## Results
The models were evaluated using:
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC
